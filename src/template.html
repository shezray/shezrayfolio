RoboSquad

A responsive and dynamic React application that fetches robot profile data from an external API and lets users search and filter through the robotic team in real-time. This project was built to demonstrate my skills in React component architecture, state management using Redux, and handling asynchronous API requests. Key features include: Live robot data fetched from an API using Redux-thunk, Real-time search functionality powered by Redux state, Scrollable list UI with error boundaries for better UX, Clean and modular code structure using React best practices. By combining Redux for global state control and ErrorBoundary for error handling, RoboSquad is a great example of building scalable, user-interactive front-end applications with robust data flow.
